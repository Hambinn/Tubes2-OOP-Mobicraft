## Technologies Used
- Java
- Java Swing


## Setup
Untuk menjalankan program, dibutuhkan gradle v7.4.2 dan JDK 17d


## Usage
1. Clone this repository (first use only)
2. Open command line and change directory to local folder repository
3. Write `./gradlew` to run program
